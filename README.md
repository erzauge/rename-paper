# reame-paper

Usage: renamepaper.rb [options] file [dir]

	renames file and moves file to dir                                                            
	if dir is not set renames file in place                                                            

	renaming:                                                            
	file ->dir/authorsFamilyNameYearTitle.pdf

        --doi [doi]
                                     rename file using doi
        --arxiv [arXiv id]
                                     rename file using arXiv id
